Data Binding AngularJS

NG-BIND{ Binding De One-Way($scope --> view)}

É usado para substituir o texto de um elemento específico de HTML com o valor passado.

EX:ng-bind="textoH1" atribuindo assim o valor do componente HTML dinamicamente.

NG-MODEL {Binding de Two-Way($scope --> view and view --> $scope).}

É usado para atribuir o valor de um Input,Textarea,Select para uma propriedade no $scope.

EX:ng-model="produto.descricao" atribuindo assim o valor do componente HTML para o objeto no $scope.

NG-INIT

É usado para definir variáveis ao iniciar a aplicação.

EX:ng-init="numero1 = 10; numero2 = 15" atribuindo assim as duas variáveis com os respectivos valores.

NG-SRC

É usado no lugar do SRC normal.

EX:Ao invés de se utilizar (img src="http://www.gravatar.com/avatar/diretorio" alt="Description").

Se Utiliza (img ng-src="http://www.gravatar.com/avatar/{{ hash }}" alt="Description")

NG-STYLE

É usado para definir modelos de CSS.

EX:ng-style="myStyle" aonde o 'myStyle' é um valor de style de css.

EX-Style: {"color":"red","background-color":""}.
